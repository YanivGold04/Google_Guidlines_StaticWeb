# Google Guidelines Onboarding Project

## Overview
This project was created to make onboarding easier and more effective for new users, team members, or employees. It delivers a structured and interactive experience based on Google’s onboarding principles, helping users quickly understand essential processes, tools, and expectations.

You can try the live version here:  
[**Launch the Onboarding Experience**](https://yanivgold04.github.io/Google_Guidlines_StaticWeb/standalone/index.html)

---

## Purpose
The goal of this project is to simplify and enhance the onboarding process by:
- Presenting information in a clear and engaging format  
- Reducing repetitive explanations and confusion  
- Providing a consistent onboarding experience  
- Supporting better knowledge retention and engagement  

---

## Technologies Used
- **HTML5** – for semantic and accessible structure  
- **CSS3** – for modern layout and styling  
- **JavaScript** – for interactivity and user flow  
- **GitHub Pages** – for hosting and deployment
